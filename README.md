# CustSrvc.Users
IndiFly365 - Customers API service for Customer Users.


## Getting started

Clone an existing Git repository with the following command:

1. cd ../workspace/IndiFly365/Services/Prod/Customer/
2. Clone the repo using following git command: `git clone https://github.com/MetrixIT-Solutions/CustSrvc.Users.git`
3. Go to the appropriate directory to start the Api service: `cd CustSrvc.Users`
4. Node Version >=18.12.1
5. Run `yarn` or `npm install` to install dependencies
6. Run the command: `npm run start`
7. You can find the sevice status on browser with the url `http://localhost:3105 or http://localhost:3115`

Push an existing Git repository with the following command:

```
cd existing_repo
git remote add origin https://github.com/MetrixIT-Solutions/CustSrvc.Users.git
git branch -M main
git push -uf origin main
```
